List select

命名空间：Ext.lt.HTML.select

说明：

下拉框应用
var selectobj=new Ext.lt.HTML.select(
selectobj.draw();

初始化参数说明

参数名 数据类型 说明
id String 下拉框ID
data Array 数据集
value String 下拉框显示默认值
nullvalue boolean 空值是否显示，默认为true
style String 下拉框样式
className string 下拉框样式,默认为空
valuefield string 下拉框显示数据对应的参数
textfield string 按钮点击显示数据对应的参数

方法说明

draw

将下拉框显示到页面中，传入的DIV对象将作为窗口的内容显示
selectobj.draw(DivId)

getId

获取下拉框id
selectobj.getId()

getValue

返回下拉框默认值。即value
selectobj.getValue()

setValue

设置下拉框默认值
selectobj.setValue(value)

getText

显示当前下拉框显示内容
selectobj.getText()

事件：

onchange

当选择框内容发生改变时产生,参数为window.event对象。该事件this对象指向select标签
selectobj.onchange=function(){}

onselected(data,value,text)

当选择框内容被选择是产生,参数为数据集对象，选中选项的值，选中选项的文本。
selectobj.onselected=function(data,v,t){}
