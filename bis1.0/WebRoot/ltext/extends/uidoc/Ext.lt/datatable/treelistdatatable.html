treelistdatatable 插件

说明：

treelistdatatable 插件是一个特殊的列定义，用来实现treelist功能

初始化参数说明

参数名 数据类型 说明
columns String/Array[String] 列名或列名的数组，用于生成tree效果的列名，必须按层次顺序传。被设置的列将自动隐藏。可以修改hiddencolumn属性，或通过调用table的setHiddenColumn接口控制这些列的显示
levelcolumn String 指定数据集中用于表示数据级次的属性名，默认‘level’，该属性必须存数字。
autolevel boolean 自动处理级次，默认值为false，如果为true则在列对象初始化时自动计算每条数据的级次。计算规则为按提供的列对象顺序第一个值为null的字段即为级次
rootlevel int 根级数据的级次编码，默认为0
treeaction boolean 是否生成展开、折叠动作，默认为false数据全部显示，如果设置为true，则需要指定数据折叠状态字段、初始显示层次等属性
statcolumn String 记录数据展开/折叠状态的字段名，默认为'_stat'。该字段值为open或close
isleafcolumn String 标记该数据是否为末级数据的字段名，默认为'_isleaf'。当属性值为true时表示没有下级数据，否则将在单元格前绘制展开、收起图标
initlevel int 大于0的整数，默认值为0.当treeaction为true时，该属性生效并用于控制表格初始状态时显示到哪一级数据
hiddencolumn boolean 默认值true，控制是否显示用于生成tree的列。

属性说明

version

组件版本号，3.0或3.5

type

组件类型，datatable

方法说明

setCols

设置表格列对象，数组形式。如果不设置，组件将通过数据集自动创建列设置。datatable中已经默认实现编码、单选、多选列，可以通过columns属性直接使用

table.setCols([
	table.columns.seq,
	table.columns.checkbox,
	{name:'code',alias:'用户编码',type:'S'}
]);

getCols

获取DataTable列对象集合，返回数组
datatble.getCols();

getCol

获取DataTable列对象
datatble.getCol(1);

draw

将表格渲染到指定HTML对象中，一般为DIV
datatable.draw(HTMLElement)

position

按照表格生成的HTML元素的大小，重新计算表格内个区域的位置
datatable.position()

// "bdgmanagedivision","bdgagency","expfunc1","expfunc2","expfunc3","billcode","sum(amt)"

var table=new Ext.lt.datatable35(cubedata);
table.setCols([
table.columns.seq
,new table.treelistcolumn({
		alias:'合并列'
		,width:200
		,columns:['bdgmanagedivision','bdgagency','expfunc1','expfunc2','expfunc3','billcode']
		,autolevel:true
		,treeaction:true
		,initlevel:3
		//,hiddencolumn:false
	})
,{name:'bdgmanagedivision',alias:'部门',fn:function(i,j,rs,value){return rs['bdgmanagedivision']==null?'总计':value}}
,{name:'bdgagency',alias:'单位',fn:function(i,j,rs,value){return rs['bdgmanagedivision']==null?value:rs['bdgagency']==null?'小计':value}}
,{name:'expfunc1',alias:'类'}
,{name:'expfunc2',alias:'款'}
,{name:'expfunc3',alias:'项'}
,{name:'billcode',alias:'单号'}
,{name:'sum(amt)',alias:'金额',type:'N'}
]);
table.setMouselight('#597EAA');
table.mousedrag(false);
table.setClassName('dttheme_ifmis');
table.draw(treelist_demo)
