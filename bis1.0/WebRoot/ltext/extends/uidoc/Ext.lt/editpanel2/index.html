editpanel2

命名空间：Ext.lt.editpanel2

说明：

标准的JS表单对象，通过创建JS对象实现表单区的功能。

支持类型

类型标识 类型说明 样式 API
input 文本编辑框 文本框组件API
ti 文本区间编辑框 文本框区间组件API
amtinput 数字编辑框 数字输入框组件API
_amtinputi 数字区间编辑框 数字输入区间API
treeselect 下拉树编辑框 下拉树编辑框组件API
date 日期编辑框 日期组件API
di 日期区间编辑框 日期区间组件API
textarea 大文本域 大文本域组件API
自定义组件 自定义组件



自定义组件API

创建对象：

new Ext.lt.editpanel2(cfg); 配置信息以参数的形式传入

editpanel2传入参数属性：

属性名 数据类型 赋值举例 说明
columnsize 数字 3 每行显示组件个数，默认为3，会根据浏览器窗口大小调整
hasconfig 布尔值 true 是否生成配置按钮，默认不生成
hasconfigobj JS对象

{ title:'配置查询条件', clickHandle:configConditionButton  }

定义配置按钮的title与按钮事件，如不生成配置按钮，该属性可不指定
configs* js数组 包括展示在表单中每一个表单组件的对象，详见表单组件对象属性介绍

表单组件对象公有属性：

属性名 数据类型调 说明
name 字符串 "remark" 英文标识
label 字符串 摘要 中文显示名称
isrequired 布尔值 true 是否必填，默认为false
labelwidth 数字 120 指定中文显示名称label宽度，默认120px
type 字符串 input(对应文本框) 组件类型,请参考 editpanel2支持的编辑组件类型

editpanel2对外接口：

方法名 参数 返回值 说明
clear() 依次调用各个组件的clear方法，清空表单
draw(el) el:HTML上的DIV或SPAN 块区域（需预设width） 符合前端组件规范的draw方法，将组件绘制到el
resize(w,h) w:宽度(像素) h:高度(像素) 符合前端组件规范的resize方法，重设editpanel宽高，内部组件布局的计算会在该方法内完成。
bind(obj) JS对象 表单区赋值并绑定，初次新建editpanel，需调用_edit.bind()。
 e.g. var object = {
  exfunc:{
    val:'12248',
    sval:'002-测试022'
 },
   remark:"测试的"
 }
  var tagObj={
   jsonval:object
  }
  _edpanel.bind(tagObj); //同步表单区的值
check() 布尔值 依次调用各个组件的check方法，有且仅当所有组件的check()方法返回true时，该方法返回true

editpanel2变量：

变量名 类型 说明
values JS对象 返回当前表单的值集

editpanel2消息：

消息 传入对象 说明
Ext.lt.message.send("update", "updateform", obj); obj为JS对象 格式：
{src:_edpanel2, //editpanel对象
   name:"remark", //单个组件name
   value: document.getElementById("area").value //存储的值
});修改表单存储的单个组件的值。
