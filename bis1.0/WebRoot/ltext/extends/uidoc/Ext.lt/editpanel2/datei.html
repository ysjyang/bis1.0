日期区间组件：

editpanel2提供标准日期区间组件类型，支持输入指定格式的日期区间。

属性

属性名 数据类型 赋值举例 说明
format 字符串 "%Y%m%d" 日期格式，默认为%Y-%m-%d'

日期区间组件 
	var cfg = {
			columnsize:1,
			configs:[{
				name:"date",
				label:"日期区间",
				type:"di",
				isrequired:true

				}]
			}
		 var _edpanel = new Ext.lt.editpanel2(cfg);
		 var el = document.getElementById("editform");
